Download do Backend (API) - ERP Compras

Baixe e execute o backend localmente para que a interface web mostre o status Online.

⬇️ Baixar backend.zip

Contém: servidor.py, iniciar_servidor.sh, dados de exemplo.

Como rodar

1) Descompacte o zip em qualquer pasta

2) No terminal, dentro da pasta descompactada, rode:

python3 servidor.py

Ou com o script:

bash iniciar_servidor.sh

O servidor sobe na porta 8080 com CORS liberado.

Conectar a interface

Abra a interface hospedada no GitHub Pages informando a URL da sua API:

https://sanderrosa.github.io/POO-PROJETO-FINAL/?api=http://SEU_HOST:8080/api
  • SEU_HOST pode ser o IP da sua máquina na LAN (ex.: http://192.168.0.12:8080/api).
  • Se quiser expor pela internet, use um túnel (ngrok, localhost.run) e passe a URL pública.
  • Depois de acessar uma vez, a URL fica salva no navegador.

Endpoints disponíveis

  • GET /api/status
  • GET /api/fornecedores
  • GET /api/ordens
  • GET /api/estoque
  • GET /api/financeiro
  • POST /api/fornecedores
  • POST /api/ordens
Funciona com Python 3 (sem dependências extras). Se precisar, fale comigo!
